RECORDING TIP

Make sure pg_stat_statements is enabled in PostgreSQL before recording. Run: CREATE EXTENSION IF NOT EXISTS pg_stat_statements; and restart Postgres. Then run the slow query a few times first so it shows up.

SCENE 6 · 2:45 – 3:20 ★ KEY SCENE — Index Advisor + Fix

Index Advisor.
The fix, ready to run.

ON SCREEN

  • → Click "Index Advisor" tab
  • → Paste the same query in the SQL box
  • → Click "Analyze Indexes"
  • → Results appear — two sections:
  •    • Missing indexes from catalog (FK columns with no index)
  •    • Inferred from query — WHERE clause: status, created_at
  • → Show the DDL recommendation:
  • CREATE INDEX idx_orders_status_created
    ON orders (status, created_at DESC);
  • → Click "Copy" button — clipboard icon

SAY THIS
