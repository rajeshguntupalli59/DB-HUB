Frequently asked questions

What do I need to run DB Hub?

A server (Linux, Windows, or Mac) with Python 3.11+ installed. DB Hub runs as a single FastAPI process — no Docker required, no external database server. A SQLite file is created automatically on first run.

Do I need an Anthropic API key?

Only for the AI Documentation and AI Assistant modules. The other 6 modules (Schema Browser, ER Diagram, Query Optimizer, Change Tracker, Connections, Team) work with no AI key at all. When you do use AI, DB Hub uses your own key — Anthropic bills you directly at their standard rates.

Can I connect multiple databases at once?

Yes. You can save unlimited named connections — a mix of PostgreSQL, SQL Server, and MySQL — and switch between them with a single click inside any module.

What's the difference between the license tiers?

All tiers include all 8 modules and all 3 databases — there are no feature gates. The difference is: Starter = 1 server, up to 5 users. Team = 1 server, unlimited users. Business = unlimited servers (useful for agencies running separate installs per client or per environment).

Can I upgrade my license later?

Yes. Email us and we'll apply your original purchase price as a credit toward the higher tier.

Is there a refund policy?

14 days, no questions asked. If DB Hub doesn't work in your environment and we can't fix it, we'll refund you in full.
